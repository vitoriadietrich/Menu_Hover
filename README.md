# Navigation Menu Hover Animation in HTML & CSS | CSS Hover Animation

Um projeto simples de **menu de navegação com animação no hover** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar **efeitos de hover em menus de navegação**, deixando a interface mais interativa e moderna.

## 🛠️ Tecnologias

- HTML
- CSS (hover, transition)

## 🎯 Funcionalidades

- Animação ao passar o mouse
- Menu estilizado
- Design simples e responsivo

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
